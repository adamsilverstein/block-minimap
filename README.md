## Requirements

* PHP 5.6+
* [WordPress](http://wordpress.org/) 5.2+

## Installation

1. Install the plugin via the plugin installer, either by searching for it or uploading a .zip file.
1. Activate the plugin.
1. Use Block Minimap!
